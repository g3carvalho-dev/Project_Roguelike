# Project Roguelike

Repositório do Projeto Roguelike — um jogo desenvolvido do zero com **C#** e **Unity**.

Este projeto é uma demonstração de jogo desenvolvida para a disciplina de Programação de Jogos Digitais, ministrada pelo professor Denis.

---

## Sobre o Projeto

Project Roguelike é um jogo no estilo *roguelike* criado com Unity, onde o jogador explora masmorras geradas proceduralmente, enfrentando inimigos e coletando itens em cada nova tentativa.
